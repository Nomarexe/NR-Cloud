Benvenuti al NR Cloud

Questo e un progetto di servizzio cloud che permette di caricare e visualizzare file multimediali come immagini, video e audio in modo dinamico. Il progetto è realizzato con Python e utilizza un server web per gestire le richieste e servire i contenuti.

Il progetto è progettato per essere semplice e intuitivo, permettendo agli utenti di caricare file nelle rispettive cartelle (Galleria, Video, Audio) e visualizzarli automaticamente senza la necessità di ricaricare manualmente la pagina. Il server e impostato nella porta 8080 e può essere avviato con il comando python3 server.py. Una volta avviato, puoi accedere al servizio tramite il tuo browser all'indirizzo http://localhost:8080.

Esplora le sezioni:

  • Galleria di immagini - Una raccolta di immagini e video che si aggiorna automaticamente quando carichi nuovi file nella cartella "Galleria".
  • Video - Sezione video dinamica che si aggiorna automaticamente quando carichi nuovi file nella cartella "Video".
  • Audio - Sezione audio che si aggiorna automaticamente con i file caricati nella cartella "Audio".
  • Documenti - Sezione documenti per visualizzare e gestire file PDF, DOCX, TXT, SVG e altri formati.
